# Changelog

## 1.0.0 - 2026-07-03

- Added portable builder for Rebex Tiny Web Server.
- Added automatic official Rebex ZIP download.
- Added automatic Rebex-compatible PFX generation with generated password.
- Added automatic `RebexTinyWebServer.exe.config` configuration for ports 80 and 443.
- Added administrator startup launcher.
- Added client Root CA installation package generation.
- Added interpreted Python entry point and compiled Windows executable.
